Release highlights

Joomla! 3.5 compatibility and various bug fixes.

PHP 5.4.0+ and Joomla! 3.4+ is required

This version requires PHP 5.4.0+ (recommended), 5.5, 5.6 or 7.0 (PHP 7.0 only supported on Joomla! 3.5). It won't install on hosts running PHP 5.3.29 or earlier, including PHP 5.2, 5.1, 5.0 and all 4.x versions. Moreover, this release will only work on Joomla! 3.4 and later. We strongly recommend using the latest Joomla! 3.x version released by the Joomla! Project.
