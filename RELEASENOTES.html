Release highlights

Workaround for CLI scripts. Joomla! 3.4.7 introduced a major bug which made it impossible for third party CLI scripts, used in CRON jobs, to complete successfully. Unfortunately it also affected the CLI scripts of our own software. This release applies a workaround which lets the CLI scripts execute without a problem.

Rewritten using FOF 3. The component is now 30% faster on average.

PHP 5.4.0+ and Joomla! 3.4+ is required

This version requires PHP 5.4.0+ (recommended), 5.5 or 5.6. It won't install on hosts running PHP 5.3.29 or earlier, including PHP 5.2, 5.1, 5.0 and all 4.x versions. Moreover, this release will only work on Joomla! 3.4 and later. We strongly recommend using the latest Joomla! 3.x version released by the Joomla! Project.
